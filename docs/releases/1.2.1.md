# Release 1.2.1

## Packages

| Package | Version |
| ------- | ------- |
| `rxfy` | `1.2.1` |
| `rxfy-react` | `1.2.1` |

## Highlights

A documentation-only patch release. The `rxfy` and `rxfy-react` package READMEs were rewritten to match the docs site — a new lockup logo, a clearer intro, and the three design principles behind rxfy — with no changes to runtime code. No API, behavior, or dependency changes.

## Changes

### Documentation

- **`rxfy` README rewrite.** Replaced the plain title with the rxfy lockup logo (dark/light variants) and rewrote the intro to mirror the docs site: a one-line description with pronunciation, the three core principles (one slot per entity, slots as RxJS Observables, declared fetch/model/mutations with normalization and SSR), and links to the [normalization](https://rxfy.vanya2h.me/core-concepts/normalization) and [why rxfy?](https://rxfy.vanya2h.me/why) pages. ([`packages/rxfy/README.md`](../../packages/rxfy/README.md))

- **`rxfy-react` README logo.** Swapped the plain title for the same rxfy lockup logo (dark/light variants). ([`packages/rxfy-react/README.md`](../../packages/rxfy-react/README.md))
