# Wrapped

`IWrapped` is the discriminated union for async state, the `IDLE / PENDING /
FULFILLED / REJECTED` pattern used throughout rxfy. It is what the query cache holds
per key (each query is an `Atom<IWrapped<…>>`) and what the [`usePending`](/react#usepending)
hook returns. Construct values with the helpers:

```ts
import { createIdle, createPending, createFulfilled, createRejected } from "rxfy";

createIdle<string>();     // { type: "IDLE" }
createPending<string>();  // { type: "PENDING" }
createFulfilled("hi");    // { type: "FULFILLED", value: "hi" }
createRejected("oops");   // { type: "REJECTED", error: "oops" }
```

## The type

`IWrapped` is a discriminated union keyed on `type` (a `StatusEnum` value). Narrow on
`type` to access `value` (only on `FULFILLED`) or `error` (only on `REJECTED`):

```ts
enum StatusEnum {
  IDLE = "IDLE",
  PENDING = "PENDING",
  FULFILLED = "FULFILLED",
  REJECTED = "REJECTED",
}

type IWrapped<T> =
  | { type: StatusEnum.IDLE }
  | { type: StatusEnum.PENDING }
  | { type: StatusEnum.FULFILLED; value: T }
  | { type: StatusEnum.REJECTED; error: unknown };
```

The optional second type parameter narrows the union to specific states, e.g.
`usePending` returns `IWrapped<T, StatusEnum.PENDING | StatusEnum.FULFILLED | StatusEnum.REJECTED>`
(never `IDLE`).

## Helpers

| Helper | Returns |
|---|---|
| `createIdle<T>()` | `{ type: "IDLE" }` |
| `createPending<T>()` | `{ type: "PENDING" }` |
| `createFulfilled<T>(value)` | `{ type: "FULFILLED", value }` |
| `createRejected<T>(error)` | `{ type: "REJECTED", error }` |

## Signatures

```ts
function createIdle<T>(): IWrapped<T, StatusEnum.IDLE>
function createPending<T>(): IWrapped<T, StatusEnum.PENDING>
function createFulfilled<T>(value: T): IWrapped<T, StatusEnum.FULFILLED>
function createRejected<T>(error: unknown): IWrapped<T, StatusEnum.REJECTED>
```
