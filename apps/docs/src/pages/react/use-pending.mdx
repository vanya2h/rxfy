# usePending [The status value behind Pending]

The hook behind [`Pending`](/react/pending), for when you need the status value rather than
rendered UI. It subscribes to any `ObservableLike<T>` (an `Observable<T>` or a plain value) and
returns the current `IWrapped<T>`, narrowed to `PENDING` / `FULFILLED` /
`REJECTED`, never `IDLE`. Pass `getDefaultValue` to start `FULFILLED` instead of `PENDING`.

```tsx
import { usePending, useStateData } from "rxfy-react";
import { StatusEnum } from "rxfy";
import { fetchTodos, todosState } from "./todos";

function TodoCount({ params }: { params: object }) {
  const { data$ } = useStateData({ state: todosState, fetchFn: fetchTodos, params });
  const wrapped = usePending(data$);

  if (wrapped.type === StatusEnum.PENDING) return <span>…</span>;
  if (wrapped.type === StatusEnum.REJECTED) return <span>error</span>;
  return <span>{wrapped.value.todos.length} todos</span>;
}
```

```ts
function usePending<T>(
  source$: ObservableLike<T>,
  getDefaultValue?: () => T,
): IWrapped<T, StatusEnum.PENDING | StatusEnum.FULFILLED | StatusEnum.REJECTED>

type ObservableLike<T> = Observable<T> | T;
```

The `IWrapped<T, K>` it returns is the async-status discriminated union, narrowed
to the `K` cases:

```ts
type IWrapped<T, K extends StatusEnum = StatusEnum> = {
  [StatusEnum.IDLE]: { type: StatusEnum.IDLE };
  [StatusEnum.PENDING]: { type: StatusEnum.PENDING };
  [StatusEnum.FULFILLED]: { type: StatusEnum.FULFILLED; value: T };
  [StatusEnum.REJECTED]: { type: StatusEnum.REJECTED; error: unknown };
}[K] & { type: K };
```

The matching `createIdle` / `createPending` / `createFulfilled` / `createRejected`
constructors are exported from `rxfy` for building these values directly.

:::warning
`source$` must be referentially stable across renders; wrap inline-created sources in
`useMemo`.
:::
