# Guides [Task-focused walkthroughs of common rxfy patterns]

End-to-end walkthroughs that put the [Core Concepts](/core-concepts) and
[React Bindings](/react) together on real features.

- [Server-Side Rendering](/core-concepts/ssr) — render with data on the server and hydrate it on the
  client: on-demand fetching, `dehydrate`/`hydrate`, and streaming SSR.
- [Pagination and infinite scroll](/guides/pagination) — load a list one page at a time and
  append each page into one growing list with [`useStatePagedData`](/react/use-state-paged-data).

Building your first app or turning on real-time sync? Those now live in
[Getting Started](/getting-started): [Create Store](/getting-started/create-store) covers models,
states, mutations, and forms end to end, and [Add Sync Client](/getting-started/add-sync-client)
wires the live server and browser Sync Client.
